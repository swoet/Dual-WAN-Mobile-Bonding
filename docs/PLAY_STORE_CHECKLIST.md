# Play Store Publishing Checklist

- [ ] App identity: title, short & full description, screenshots, icon
- [ ] Content rating questionnaire
- [ ] Privacy policy URL (link to repo PRD/Privacy Policy as appropriate)
- [ ] Permissions disclosure for VpnService, networking, foreground service
- [ ] Data safety form (explain telemetry opt-in and aggregated counters)
- [ ] Release tracks: internal testing first, then closed, then production
- [ ] Signed release APK / App Bundle uploaded
- [ ] Changelog / Release notes
- [ ] Target/Min SDK compliance (target 33, min 28)
- [ ] Device and form factor listings
